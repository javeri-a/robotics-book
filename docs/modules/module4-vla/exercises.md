# Exercises

## Exercise 1: Vision-Language System

Create a system that combines visual perception with language understanding.

### Requirements:
- Use a vision-language model for object detection
- Process natural language queries about objects
- Display results with bounding boxes and text
- Evaluate accuracy on standard datasets

## Exercise 2: Language-Guided Manipulation

Implement a robot system that executes manipulation tasks based on language commands.

### Requirements:
- Integrate LLM with robot control system
- Map language commands to manipulation primitives
- Execute tasks in simulation environment
- Handle ambiguities in language commands

## Exercise 3: Complete VLA Pipeline

Build a complete Vision-Language-Action pipeline for a specific task.

### Requirements:
- Combine all three VLA components
- Implement real-time processing
- Test with human users providing natural language commands
- Evaluate task success rate and user satisfaction
