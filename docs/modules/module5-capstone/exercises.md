# Exercises

## Exercise 1: Basic Voice Command

Implement a basic voice-controlled robot that responds to simple commands.

### Requirements:
- Set up audio input and speech recognition
- Implement basic command recognition
- Control robot movement with voice
- Provide audio feedback to the user
- Test in a controlled environment

## Exercise 2: VLA Integration

Combine vision, language understanding, and action execution.

### Requirements:
- Use camera to identify objects
- Process natural language requests about objects
- Execute manipulation or navigation tasks
- Handle ambiguous requests
- Evaluate system accuracy

## Exercise 3: Capstone Challenge

Deploy the complete voice-controlled humanoid robot system.

### Requirements:
- Integrate all modules from the textbook
- Implement safety mechanisms
- Test with diverse voice commands
- Evaluate system performance and reliability
- Document lessons learned and potential improvements
