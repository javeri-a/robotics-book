# Exercises

## Exercise 1: Custom Message Publisher

Create a publisher that sends custom messages containing your name and the current timestamp.

### Requirements:
- Create a custom message type with a string field for the name and a time field
- Create a publisher that sends this message every second
- Include proper error handling

## Exercise 2: Service Server

Create a service that takes two integers as input and returns their sum.

### Requirements:
- Define a service interface with request and response fields
- Implement the service server
- Test with a service client

## Exercise 3: Navigation Simulation

Using Gazebo, implement a node that navigates a simulated robot to a target location.

### Requirements:
- Use ROS 2 navigation stack
- Implement obstacle avoidance
- Log navigation metrics
